Raspberry Pi LED Project

Create by:
  Jackson Sorrells
  Brandon Wood

Purpose:
  To manipulate LEDs on an LED strip using a Raspberry Pi and Python code. The strip is changed
  based on personal patterns or a color display based on the weather condition and temperature
  of a given location.

APIs Used:
  rpi_ws281x - https://github.com/jgarff/rpi_ws281x
    Used to manipulate a single LED on a strip from the Raspberry Pi
  pywapi - https://code.google.com/archive/p/python-weather-api/
    Used to pull weather information for a given location
      Includes weather condition and temperature
